this is the IFRAME test

the Iframe is like a window that has been cut into your page
and in that page you can see another window

is an abreviation for inline frame
One common use is to embed a Google maps into the page , but the content can be any html page

There are a few attributes that it must have:

  • src specifies the URL
  • height specifies the hieght if the iframe
  • width specifies the width iframe
  • scrolling indicates whether the frame should have a scroller or not
    not availabe in HTML5
  • frameborder indicates where the frame should have a border. 0 indicates nnot frame while 1 indicates a frame
