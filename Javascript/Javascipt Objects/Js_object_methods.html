JS object methods

We can add functions as properties to an object and they would be called methods A method is a function that is a property inside of an object

Why add methods to an object

  • keeps code organized so we can group things logically together
  • Avoid namespace collision
    1. we can have similar namespaces by placing them as methods to an object

The keyword THIS refers to the object it belongs to It can have different values depending on where it is used

  1. In a method it refers to the parent object
  2. in a function it refers to the global object
  3. alone, it refers to the global object
  4. in an event, this refers to the element that received the event
  5. in a function in strict mode this is undefined
