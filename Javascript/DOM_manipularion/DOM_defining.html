DOM Defining

DOM- Document Object Model- is the interface between the Javascript and the HTML and CSS.
It's Javascript objects and functions that we can use to interact with our Html and Css.

when a webpage is loaded the browser creates a Document Object Model of the page.
It is an object riented representation of the page which can be modified with a scripting language such as Javascript.

A webpage is a document, that can be either displayed in the browser window, or as a HTML source . The DOM represents that same document so it can be manipulated.
